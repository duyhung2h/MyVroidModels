【使用方法】

.Vroidファイルを開き、新規アイテムとして保存してください。
または、以下の方法でパラメーター調節をした後、
テクスチャをインポートしてください。


・ウェディングドレス 

1.衣装編集→衣装→上半身を「ワンピース(半袖)」に変更

2.パラメータを「パラメーター参考.png」を参考に変更

3.テクスチャを編集からテクスチャタブのデフォルト画像にインポート


・手袋

1.衣装編集→衣装→腕飾りで「透明」を選択

2.テクスチャを編集からテクスチャタブのデフォルト画像にインポート


・下着

1.衣装編集→衣装→上半身インナーで「ベアトップ」を選択

2.テクスチャを編集からテクスチャタブのデフォルト画像にインポート


胸が小さい場合は胸元が見えやすいので下着着用推奨です。
同梱したものと他のデータを使う場合は背中が開いているので
見えないようご自身で改変などしてください。
下半身インナー、靴などは同梱されていません。お好きなものをご着用ください


【How to use】

Open the .Vroid file and save it as a new item.
Or, after adjusting parameters in the following way
Import the textures.


・Wedding Dress 
1. Edit Costume→Costume→Change upper body to "One-piece (short sleeves)

2. Change parameters with reference to "Parameter Reference.png

3. Import textures from Edit to default image in the Textures tab


・Gloves
1. Edit Costume → Costume → Arm Decoration and select "Transparent

2. Import texture from Edit to default image in Textures tab


・Underwear
1. Edit Costume → Costume → Upper Body Innerwear and select "Bare Top

2. Import texture from Edit to default image on Texture tab


If the breasts are small, it is recommended to wear underwear as the bust is easily visible.
If you use the included data or other data, the back is open.
Please modify the image so that it is not visible.
Lower body underwear and shoes are not included. Please wear what you like.
